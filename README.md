# capybara-rspec

## Setup Project Structure

```
> cd $WORKSPACE
> mkdir capybara-rspec
> cd capybara-rspec

Generates Gemfile
> bundle init 

Generate rspec folder structure 
> rspec --init

```

## Sauce Labs Configuration



```
export SAUCE_USERNAME=your_username
export SAUCE_ACCESS_KEY=your_access_key
```
